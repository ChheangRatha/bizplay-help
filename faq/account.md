# 계정, 회원가입

## 아이디 생성 규칙

**아이디를 생성하는 방법은 2가지가 있습니다.**

1. **회사가 직원정보를 생성할 때 직원 별 아이디를 지정하는 방법**
2. **직원이 직접 회원가입하는 방법**

## 비밀번호 변경 주기 / 규칙

* **비밀번호는 영문/숫자/특수문자 8자리 이상 \(영문, 숫자는 10자리 이상\)으로 사용 \(제외문자 : +, %, \#, &, ', ", \, &lt;, &gt;\)**
* **연속적인 3자리 숫자 및 아이디와 동일한 비밀번호는 사용할 수 없습니다.**
* **최근 사용한 비밀번호는 다시 사용할 수 없습니다.**
