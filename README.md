# 지식창고가 무엇인가요?

안녕하세요. 이 곳은 상품들에 대한 팁이나 이용 가이드, 자주 묻는 질문을 모아 알려드리는 공간입니다. 아래와 같은 내용을 주로 다룹니다.

* 미리 알고 있으면 좋은 팁, 주요 기능
* 자주 묻는 내용, 케이스 별 조치 방법
* 상품과 관련된 정책, 관련 규정, 보안 정책 등
